# `BME – Client-Side Systems – React Homework Project`

<h3 align="center">This project was created as the React-based homework assignment for the
**Client-Side Systems** course at the Budapest University of Technology and Economics (BME).</h3>

## Requirements

The original assignment specification can be found in /docs/requirements.pdf.

## Project Description

The task was to develop a unique React application based on a self-written specification and the requirements defined by the department.
The project demonstrates component-based development, the use of React hooks, and modern frontend practices.

## Technologies Used

- React
- TypeScript
- HTML, CSS
- npm / Vite

## Usage

- `npm run dev` - Starts a dev server at http://localhost:5173/

- `npm run build` - Builds for production, emitting to `dist/`

- `npm run preview` - Starts a server at http://localhost:4173/ to test production build locally
